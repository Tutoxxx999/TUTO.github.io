Inicio

En esta página exploraremos los fundamentos de la programación utilizando el lenguaje C, desde los tipos de datos hasta algoritmos de ordenamiento.

Lenguaje C: Fundamentos

El lenguaje C es un lenguaje de programación de propósito general, conocido por su eficiencia y control de bajo nivel.

Ejemplo:

#include <stdio.h> int main() { printf("¡Hola, mundo!\n"); return 0; }

Datos: Tipos y Características

Los tipos de datos básicos en C incluyen:

  • int: Números enteros
  • float: Números decimales
  • char: Caracteres individuales

Ejemplo:

#include <stdio.h> int main() { int edad = 25; float altura = 1.75; char inicial = 'A'; printf("Edad: %d, Altura: %.2f, Inicial: %c\n", edad, altura, inicial); return 0; }

Estructuras de Control: Condicionales

Las estructuras de control permiten tomar decisiones en el código.

Ejemplo:

#include <stdio.h> int main() { int numero = 10; if (numero > 5) { printf("El número es mayor que 5\n"); } else { printf("El número es menor o igual a 5\n"); } return 0; }

Estructuras Repetitivas: Bucles

Los bucles permiten ejecutar un bloque de código varias veces.

Ejemplo:

#include <stdio.h> int main() { for (int i = 0; i < 5; i++) { printf("Iteración %d\n", i); } return 0; }

Arreglos: Colecciones de Datos

Un arreglo es una colección de elementos del mismo tipo.

Ejemplo:

#include <stdio.h> int main() { int numeros[5] = {1, 2, 3, 4, 5}; for (int i = 0; i < 5; i++) { printf("Elemento %d: %d\n", i, numeros[i]); } return 0; }

Métodos de Ordenamiento: Algoritmos

El ordenamiento por burbuja es un algoritmo básico para ordenar elementos.

Ejemplo:

#include <stdio.h> void bubbleSort(int arr[], int n) { for (int i = 0; i < n - 1; i++) { for (int j = 0; j < n - i - 1; j++) { if (arr[j] > arr[j + 1]) { int temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } } } } int main() { int arr[5] = {5, 3, 8, 1, 2}; bubbleSort(arr, 5); for (int i = 0; i < 5; i++) { printf("%d ", arr[i]); } return 0; }
