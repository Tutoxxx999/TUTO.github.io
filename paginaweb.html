Historia de las Computadoras

Desde los primeros dispositivos mecánicos como el ábaco, hasta los modernos sistemas digitales, la evolución de las computadoras ha revolucionado el mundo. A lo largo del tiempo, se han desarrollado grandes hitos como las computadoras ENIAC y las microcomputadoras personales.

Clases o Tipos de Computadoras

Existen varios tipos de computadoras, como las de escritorio, portátiles, servidores y supercomputadoras. Cada una está diseñada para cumplir diferentes propósitos y capacidades.

Case: Características y Tipos

Los cases son las carcasas que contienen los componentes internos de una computadora. Pueden ser de varios tamaños, como ATX, Micro-ATX y Mini-ITX, dependiendo de la motherboard que soporten.

Motherboard: Características y Tipos

La motherboard es el circuito principal que conecta todos los componentes de la computadora. Sus factores de forma más comunes incluyen ATX, Micro-ATX y Mini-ITX.

Ranuras de Expansión

Las ranuras de expansión, como PCI y PCIe, permiten instalar tarjetas adicionales como gráficas, de red o de sonido para mejorar las capacidades del sistema.

Unidades de Almacenamiento

Las unidades de almacenamiento incluyen discos duros (HDD), unidades de estado sólido (SSD) y unidades NVMe, cada una con diferentes velocidades y capacidades.

Tarjetas de Expansión

Las tarjetas de expansión son componentes adicionales que se conectan a la motherboard para agregar funcionalidades como gráficos avanzados, conectividad de red o sonido de alta calidad.

Glosario de Términos

  • ATX: Estándar de tamaño para motherboards y cases.
  • HDD: Disco duro tradicional con partes mecánicas.
  • SSD: Unidad de estado sólido, más rápida que el HDD.
  • PCI: Ranura de expansión más antigua para tarjetas adicionales.
